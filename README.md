- 1. Open to about:config
- 2. Enable *toolkit.legacyUserProfileCustomizations.stylesheets* > about:config > (false = true)
```bash
     toolkit.legacyUserProfileCustomizations.stylesheets
```
- 3. Open about:profiles → «Root Directory» > button «Open Directory»
- 4. Move root dir with custom theme 'chrome' in Root  Directory
- 5. Move all files theme dir in . 
```bash
   mv example_dir_theme ~/.mozilla/firefox/*.default
```
